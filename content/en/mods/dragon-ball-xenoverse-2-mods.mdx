<Section icon={Shield} title="Downgrading Your Game Version via Steam Console">

When a new update drops, Steam automatically updates your local files. To restore compatibility for your **dragon ball xenoverse 2 mods**, you must manually download an older depot. This process uses Steam's built-in command-line console to request older manifests directly from the official servers.

| Component | Identifier | Purpose |
|-|-|-|
| App ID | 454650 | Identifies the main game on the Steam platform |
| Depot ID | 454651 | Identifies the specific content package to download |
| Manifest ID | Variable | Identifies the specific historical version patch |

<Steps>
  <Step num="1" title="Open the Steam Console">
    Press the Windows Key + R on your keyboard to open the Run dialog box. Type `steam://open/console` and press Enter. This action opens a dedicated console tab within your active Steam client.
  </Step>
  <Step num="2" title="Locate the Manifest ID">
    Navigate to SteamDB in your web browser. Search for App ID 454650, select the Depots tab, and choose the main content depot. Browse the history tab to find the manifest ID associated with your target version, such as version 1.25.02.
  </Step>
  <Step num="3" title="Execute the Download Command">
    In the Steam Console, enter the download command using the following syntax: `download_depot 454650 454651 [Manifest_ID]`. Press Enter to begin the background download.
  </Step>
  <Step num="4" title="Replace the Game Files">
    Once the console displays a message confirming the download is complete, navigate to the designated folder path. Copy the downloaded files and overwrite the files in your main directory.
  </Step>
</Steps>

<Callout type="info" title="Steam Download Location">
Steam downloads console depots to your default installation directory under `Steam\steamapps\content\app_454650\depot_454651`. Ensure you copy the contents of this folder, not the folder itself, into your main directory.
</Callout>

</Section>
