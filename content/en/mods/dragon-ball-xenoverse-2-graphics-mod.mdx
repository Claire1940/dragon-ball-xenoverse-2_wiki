<Section icon={Shield} title="Mod Setup: Essential Prerequisites & Compatibility">

Before you begin downloading any visual enhancements, you must prepare your system and game directory. Modding is exclusive to the PC version of the game on Steam; console versions do not support external file injections. Because the game receives periodic official updates from Bandai Namco—such as the recent Future Saga and DAIMA expansions—the modding tools must be updated in tandem. Running outdated patchers or installers on a newer game version will lead to immediate crashes.

The backbone of any modded setup consists of two primary tools: the XV2PATCHER and the XV2 Mod Installer (XV2INS). The patcher bypasses the game's file verification system, allowing it to load custom assets from a designated "data" folder instead of the locked game archives. The installer manages custom characters, skills, and outfits. For graphics mods that modify shaders or post-processing, the Reshade engine is also required, which injects a custom `dxgi.dll` or `d3d11.dll` file into the game's root directory.

| Tool Name | Core Function | Required Version | Main Risk |
| :--- | :--- | :--- | :--- |
| **XV2PATCHER** | Enables loading of external mod files | Matches current game version | Game crashes if versions do not match |
| **XV2INS** | Installs custom characters and skills | Latest release | Failed installations during major updates |
| **Reshade Engine** | Injector for custom lighting and shaders | Version 5.0 or higher | Minor compatibility issues with overlays |

Understanding how these tools interact is vital for troubleshooting. When the game launches, the patcher reads the `xv2patcher.ini` configuration file to determine which directories to load. If you install a graphics mod that replaces character models or stage textures, those files must go into the `data` folder created by the patcher. If the patcher is not running or is outdated, the game will simply ignore these files and load the default vanilla assets.

<Callout type="warning" title="Update Notice">
Whenever Bandai Namco releases an official game patch, your mods will likely stop working. Do not launch the game through Steam until the creators of XV2PATCHER release a compatible update. You can disable automatic updates in Steam to preserve your modded setup.
</Callout>

  <Step num="2" title="Install the Latest XV2PATCHER">

  <Step num="4" title="Configure the Reshade Shader Engine">
    Download the Reshade installer and run it. Select the game executable (`bin/DBXV2.exe`) and choose Direct3D 10/11/12 as the rendering API. Select the shader packages recommended by your graphics mod creator, then place the custom `.ini` preset file into the `bin` folder.
  </Step>
</Steps>

Once these steps are completed, launch the game. If you installed Reshade, you should see a small overlay bar at the top of your screen indicating that the shaders are loading. Press the designated hotkey (usually the Home key) to open the configuration menu and select your graphics preset. For texture and cel-shading mods, the changes should be visible immediately upon entering Conton City or starting a local battle.

Once your files are in place, fine-tuning your in-game settings is crucial to achieving the optimal balance between visual fidelity and performance. Some graphics mods require specific in-game configurations to display correctly. For instance, many cel-shading mods require you to disable the game's default anti-aliasing to prevent the custom black outlines from appearing blurry or pixelated.

If you encounter issues during launch, such as a black screen or an infinite loading sequence, it is usually due to a conflict between Reshade and other active overlays. Discord, GeForce Experience, and Steam overlays can interfere with the DirectX injection process. Disabling these overlays or renaming the Reshade file from `dxgi.dll` to `d3d11.dll` often resolves startup conflicts.
