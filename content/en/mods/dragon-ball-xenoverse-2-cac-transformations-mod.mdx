<Section icon={Map} title="Installation Step-by-Step Guide">

Setting up the **dragon ball xenoverse 2 cac transformations mod** requires the **XV2INS (Xenoverse 2 Installer)** tool. Because this mod adds new skills rather than replacing existing ones, you must follow the X2M installation path strictly to avoid corrupted save files.

<Steps>
  <Step num="1" title="Prerequisites">
    Download the latest version of the Xenoverse 2 Patchwork and Installer. Ensure your game version matches the mod version (currently v1.25.01 or higher).
  </Step>
  <Step num="2" title="Extract Files">
    Unzip the lazybone transformation pack. You will find several folders containing X2M files for different races and skill types.
  </Step>
  <Step num="3" title="Run XV2INS">
    Open the installer and select the X2M files you wish to add. It is recommended to install the "New Transformations" core files first before adding specific character X2Ms.
  </Step>
  <Step num="4" title="Configure Auras">
    The mod includes a separate installer for custom auras. Choose between "Vanilla Style" or "New Custom Style" auras during this step.
  </Step>
  <Step num="5" title="In-Game Purchase">
    Launch the game and head to the Skill Shop. All newly installed transformations should be available for purchase under the "Awoken Skills" tab.
  </Step>
</Steps>

<Callout type="success" title="Installation Verified">
If you see "Ultra Instinct" or "Super Saiyan 4" in the shop, the mod is correctly integrated into your game's database.
</Callout>

<Section icon={Shield} title="Revamp Mod Compatibility & Troubleshooting">

A major point of discussion in the 2026 modding community is the compatibility between lazybone's transformations and the **Revamp Mod**. Currently, these two massive mods do not play well together out of the box.

Installing "New Transformations" alongside "Revamp" often results in messed-up auras or infinite loading screens. If you install Revamp first, the auras for the new transformations may fail to display. If you install the transformations first, the game tends to crash during character selection.

| Problem | Potential Cause | Suggested Workaround |
| :--- | :--- | :--- |
| **Infinite Loading** | Conflict with Revamp's internal ID system | Choose one mod or use a compatibility patch |
| **Unknown Skill** | Missing X2M files or outdated installer | Reinstall using the latest XV2INS version |
| **Aura Glitches** | Conflicting aura ID entries | Run the Aura Installer last in your load order |
| **Tail Stretching** | Skeleton conflict with custom clothes | Remove conflicting clothing mods |
| **Combo Breaks** | 50/50 chance during transformation | Ensure "Transformed Skillset" costumes are ignored |

<Callout type="warning" title="Critical Compatibility Note">
Many users recommend uninstalling Revamp if you want a stable experience with the lazybone transformation mod, as the latter provides more direct gameplay variety for CaCs.
</Callout>
