<Section icon={Shield} title="Revertir la Versión del Juego mediante la Consola de Steam">

Cuando se lanza una nueva actualización, Steam actualiza automáticamente tus archivos locales. Para restaurar la compatibilidad de tus **mods de dragon ball xenoverse 2**, debes descargar manualmente un depósito anterior. Este proceso utiliza la consola de línea de comandos integrada de Steam para solicitar manifiestos antiguos directamente desde los servidores oficiales.

| Componente | Identificador | Propósito |
| - | - | - |
| App ID | 454650 | Identifica el juego principal en la plataforma Steam |
| Depot ID | 454651 | Identifica el paquete de contenido específico para descargar |
| Manifest ID | Variable | Identifica el parche de versión histórica específico |

<Steps>
  <Step num="1" title="Abrir la Consola de Steam">
    Presiona la tecla Windows + R en tu teclado para abrir el cuadro de diálogo Ejecutar. Escribe `steam://open/console` y presiona Enter. Esta acción abrirá una pestaña de consola dedicada dentro de tu cliente de Steam activo.
  </Step>
  <Step num="2" title="Localizar el ID del Manifiesto">
    Navega a SteamDB en tu navegador web. Busca el App ID 454650, selecciona la pestaña Depots y elige el depósito de contenido principal. Revisa la pestaña de historial para encontrar el ID del manifiesto asociado con tu versión objetivo, como la versión 1.25.02.
  </Step>
  <Step num="3" title="Ejecutar el Comando de Descarga">
    En la consola de Steam, ingresa el comando de descarga usando la siguiente sintaxis: `download_depot 454650 454651 [ID_del_Manifiesto]`. Presiona Enter para comenzar la descarga en segundo plano.
  </Step>
  <Step num="4" title="Reemplazar los Archivos del Juego">
    Una vez que la consola muestre un mensaje confirmando que la descarga se ha completado, navega a la ruta de la carpeta designada. Copia los archivos descargados y sobrescribe los archivos en tu directorio principal.
  </Step>
</Steps>

<Callout type="info" title="Ubicación de Descarga de Steam">
Steam descarga los depósitos de la consola en tu directorio de instalación predeterminado bajo `Steam\steamapps\content\app_454650\depot_454651`. Asegúrate de copiar el contenido de esta carpeta, no la carpeta en sí, en tu directorio principal.
</Callout>

</Section>

<Section icon={Swords} title="Instalación de Herramientas Esenciales de Modding">

Una vez que la versión de tu juego esté alineada con el software compatible, debes configurar el marco central que permite que se ejecuten los archivos personalizados. El ecosistema de modding se basa en dos componentes principales: el Eternity Patcher, que omite las comprobaciones de archivos, y el Mod Installer, que gestiona la integración de archivos.

<CardGrid cols={3}>
  <Card title="Eternity Patcher">
    - **Omite las comprobaciones de archivos del juego**
    - Se engancha directamente en el ejecutable
    - Esencial para todos los mods de script
  </Card>
  <Card title="XV2INS (Instalador)">
    - **Instala personajes personalizados**
    - Gestiona la asignación de habilidades y slots
    - Proporciona una interfaz de usuario visual
  </Card>
  <Card title="Editor de Guardado">
    - **Edita atributos de personajes**
    - Desbloquea objetos y rutas de misiones
    - Opcional pero muy útil
  </Card>
</CardGrid>

Para asegurar que estas utilidades funcionen correctamente, revisa las diferencias operativas específicas y los requisitos de las dos herramientas principales de modding:

| Herramienta de Utilidad | Formato de Archivo Principal | Dependencia Clave | Directorio de Instalación |
| - | - | - | - |
| Eternity Patcher | .ini / .dll | Directorio XV2Patcher | Carpeta raíz del juego |
| XV2INS | .x2m | xv2ins.exe | Cualquier lugar (Enlace a bin/xinput1_3.dll) |

Para instalar estas herramientas, extrae el archivo del Eternity Patcher directamente en tu directorio raíz. La carpeta raíz es donde reside el ejecutable de tu juego. A continuación, inicia la aplicación XV2INS y apúntala hacia el ejecutable de tu juego cuando se te solicite. Esto establece la conexión necesaria para escribir nuevos archivos en el directorio de tu juego.

<Callout type="tip" title="Copia de Seguridad de Partidas Guardadas Primero">
Antes de ejecutar el instalador por primera vez, haz una copia de tu archivo de guardado ubicado en el directorio Steam Userdata. Los cambios en la configuración de los mods ocasionalmente pueden corromper los datos de guardado.
</Callout>
