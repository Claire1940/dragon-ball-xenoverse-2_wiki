# Guía Paso a Paso para Revamp de Mods de Dragon Ball Xenoverse 2

¿Deseas mejorar tu experiencia en Dragon Ball Xenoverse 2 con mods? Esta guía te guiará a través del proceso de instalación y uso de mods para mejorar tu juego. Ya seas un modder experimentado o nuevo en la escena, este artículo te proporcionará los pasos necesarios para comenzar.

## Entendiendo el Proceso de Modding

### Verificación de Compatibilidad
Antes de sumergirte en el proceso de modding, es crucial asegurarte de que tu versión del juego sea compatible con el mod que deseas instalar. Por ejemplo, Revamp puede no ser compatible con todas las versiones de Dragon Ball Xenoverse 2. Revisa las notas de compatibilidad del mod o su sitio web para requisitos específicos.

### Copia de Seguridad del Juego
Para evitar cualquier posible pérdida de datos, siempre es una buena idea realizar una copia de seguridad de tu juego antes de realizar cualquier cambio. De esta manera, si algo sale mal durante la instalación del mod, puedes revertir al estado del juego original sin perder ningún progreso.

## Guía Paso a Paso para Instalar Mods

### Paso 1: Desinstalar Mods Existentes
Si tienes algún mod instalado, es importante eliminarlos para evitar conflictos con el nuevo mod. Esto incluye eliminar los archivos del mod de tu directorio del juego y cualquier carpeta asociada.

### Paso 2: Eliminar Archivos del Juego y Desinstalar
A continuación, necesitarás eliminar tus archivos de juego de la biblioteca de Steam y desinstalar el juego. Esto asegura una instalación limpia sin archivos restantes que podrían causar problemas.

### Paso 3: Reinstalar el Juego
Una vez que el juego esté desinstalado, puedes reinstalarlo desde Steam. Durante el proceso de instalación, asegúrate de validar los archivos para asegurarte de que todo funcione correctamente.

### Paso 4: Añadir el Patcher
Descarga el parche del mod desde una fuente confiable y extrae los archivos. Luego, navega a tu directorio del juego utilizando la aplicación Steam y localiza la carpeta "Xenoverse 2".

### Paso 5: Copiar Archivos del Patcher
Copia todos los contents de la carpeta del parche a tu directorio del juego. Esto incluirá cualquier archivo necesario para que el mod funcione correctamente.

### Paso 6: Configurar el Instalador
Abre el instalador y configura según tus preferencias. Puedes instalar mods aleatorios de personajes o habilidades para probar si el parche funciona correctamente.

### Paso 7: Iniciar el Juego
Después de configurar el instalador, inicia el juego para verificar si la versión del parche se muestra en la pantalla de inicio. Si es así, puedes proceder al siguiente paso.

### Paso 8: Extraer Lazy Bone Save Editor
Descarga el Lazy Bone Save Editor y extrae los archivos. Esta herramienta se utiliza para eliminar IDs no utilizadas y validar banderas en tu juego.

### Paso 9: Eliminar IDs No Utilizadas y Validar Banderas
Carga tu directorio del juego y el archivo de guardado en el Lazy Bone Save Editor. Elimina cualquier ID no utilizada y valida las banderas para asegurarte de que todo esté configurado correctamente.

### Paso 10: Descargar e Instalar Revamp
Descarga Revamp de una fuente confiable, extrae los archivos y abre la aplicación. Navega a la carpeta del directorio del juego y instala el mod.

### Paso 11: Iniciar el Juego desde la Carpeta Bin
Finalmente, inicia el juego desde la carpeta bin para asegurarte de que el mod esté instalado correctamente.
