<Section icon={Target} title="Asignación Óptima de Puntos de Atributo">

Con la expansión de la Casa del Gran Patriarca que permite estadísticas de hasta 125 (y futuras expansiones para el nivel 199), la asignación de puntos es más flexible que nunca. Sin embargo, la "regla de los 42 puntos" sigue siendo el estándar de oro para la gestión de Ki y Resistencia. Dado que cada 42 puntos equivalen a una barra completa, gastar puntos en incrementos de 42 asegura que no se desperdicien atributos.

### Distribución de Estadísticas Saiyan Hombre (Enfoque Nivel 199)

| Atributo | Puntos Recomendados | Razón |
| :--- | :--- | :--- |
| **Salud** | 0 - 50 | Salud base alta; confía en las QQ Bangs para el resto. |
| **Ki** | 42 u 84 | Proporciona 6-7 barras, suficiente para la mayoría de Definitivos. |
| **Resistencia** | 42 u 84 | Esencial para evasiones y maniobras defensivas. |
| **Ataques Básicos** | 125 - 200 | **Obligatorio** para builds masculinas; maximiza el daño de combo. |
| **Súper Golpes** | 0 o 105 | Elige un tipo de súper para especializarte. |
| **Súper Ráfagas de Ki** | 0 o 105 | Elige un tipo de súper para especializarte. |

### Distribución de Estadísticas Saiyan Mujer (Enfoque Ráfagas de Ki)

| Atributo | Puntos Recomendados | Razón |
| :--- | :--- | :--- |

| **Ki** | 42 | Suficiente para 5-6 barras para activar Super Vegeta. |
| **Resistencia** | 42 | Estándar para utilidad defensiva. |

<Section icon={Sparkles} title="Transformaciones y Habilidades Despertadas">

La elección de la transformación define la utilidad de tu Saiyan. Mientras que el Super Saiyan 3 era la opción tradicional para el daño físico, el meta de 2026 ha introducido opciones más especializadas. "Poder para Superar" es actualmente la favorita por sus mecánicas de supervivencia, pero "Super Vegeta" y "Bestia" siguen siendo de primer nivel para la salida de daño puro.

<Steps>
  <Step num="1" title="Desbloquear Potencial">
    Completa las pruebas de progreso en el Instituto Estrella Naranja para desbloquear Potencial Desatado, una excelente transformación equilibrada.
  </Step>
  <Step num="2" title="Dominar Formas Saiyan">
    Entrena con Vegeta en Capsule Corp para desbloquear Super Saiyan 1-3 y Super Vegeta 1-2 al alcanzar umbrales específicos de nivel y amistad.
  </Step>
  <Step num="3" title="Adquirir Poder para Superar">
    Completa las misiones de historia del DLC Future Saga Capítulo 4 para desbloquear la última transformación del meta.
  </Step>
  <Step num="4" title="Probar el Overdrive">
    Entra en el Modo Entrenamiento para practicar el tiempo de activación del Overdrive, ya que requiere condiciones de salud específicas.
  </Step>
</Steps>
