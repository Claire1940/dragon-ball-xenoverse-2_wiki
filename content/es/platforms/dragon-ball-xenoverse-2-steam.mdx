<Section icon={Target} title="Pasos para Volver a una Versión Anterior (Downgrade)">

Sigue estos pasos para revertir la versión de tu juego a un estado estable. Este proceso requiere el uso del comando 'Ejecutar' de Windows para interactuar directamente con la consola de Steam.

<Steps>
  <Step num="1" title="Abrir la Consola de Steam">
    Presiona Win+R en tu teclado, escribe `steam://open/console` y pulsa Enter para abrir la pestaña oculta de la consola de Steam.
  </Step>
  <Step num="2" title="Identificar el Manifiesto">
    Visita el sitio web de SteamDB para encontrar el número de parche específico o el ID de manifiesto que corresponda a tu versión preferida del juego.
  </Step>
  <Step num="3" title="Ejecutar Descarga">
    En la consola, introduce `download_depot 454650 454651 [IDdelManifiesto]` para comenzar a descargar los archivos específicos del juego.
  </Step>
  <Step num="4" title="Reemplazar Archivos">
    Navega hasta tu carpeta SteamApps/content para localizar los archivos descargados y sobrescribir tu instalación actual del juego.
  </Step>
</Steps>

<Callout type="warning" title="Copia de Seguridad Requerida">
Crea siempre una copia de seguridad de tus carpetas actuales `bin` y `data` antes de sobrescribir archivos para evitar perder el progreso o datos personalizados.
</Callout>

</Section>
