<Section icon={Target} title="Passos para Fazer Downgrade de Versão">

Siga estes passos para reverter a versão do seu jogo para um estado estável. Este processo requer o uso do comando 'Executar' do Windows para interagir diretamente com o console do Steam.

<Steps>
  <Step num="1" title="Abrir o Console do Steam">
    Pressione Win+R no seu teclado, digite `steam://open/console` e aperte Enter para iniciar a aba oculta do console do Steam.
  </Step>
  <Step num="2" title="Identificar o Manifest">
    Visite o site SteamDB para encontrar o número do patch específico ou ID do manifest que corresponde à versão do jogo de sua preferência.
  </Step>
  <Step num="3" title="Executar o Download">
    No console, insira `download_depot 454650 454651 [ManifestID]` para iniciar o download dos arquivos específicos do jogo.
  </Step>
  <Step num="4" title="Substituir Arquivos">
    Navegue até a pasta SteamApps/content para localizar os arquivos baixados e substituir a sua instalação atual do jogo.
  </Step>
</Steps>

<Callout type="warning" title="Backup Necessário">
Sempre crie um backup das suas pastas `bin` e `data` atuais antes de substituir os arquivos para evitar a perda de progresso ou dados salvos personalizados.
</Callout>

</Section>
