<Section icon={Shield} title="Fazendo o Downgrade da Versão do Jogo via Console Steam">

Quando uma nova atualização é lançada, a Steam atualiza automaticamente seus arquivos locais. Para restaurar a compatibilidade dos seus **mods de dragon ball xenoverse 2**, você deve baixar manualmente um depósito (depot) mais antigo. Esse processo usa o console de linha de comando integrado da Steam para solicitar manifestos antigos diretamente dos servidores oficiais.

| Componente | Identificador | Propósito |
|-|-|-|
| App ID | 454650 | Identifica o jogo principal na plataforma Steam |
| Depot ID | 454651 | Identifica o pacote de conteúdo específico para baixar |
| Manifest ID | Variável | Identifica o patch de versão histórica específico |

<Steps>
  <Step num="1" title="Abra o Console Steam">
    Pressione as teclas Windows + R no seu teclado para abrir a caixa de diálogo Executar. Digite `steam://open/console` e pressione Enter. Essa ação abrirá uma aba dedicada do console dentro do seu cliente Steam ativo.
  </Step>
  <Step num="2" title="Localize o ID do Manifesto">
    Navegue até o SteamDB no seu navegador de internet. Pesquise pelo App ID 454650, selecione a aba Depots e escolha o depósito de conteúdo principal. Navegue pela aba de histórico para encontrar o ID do manifesto associado à sua versão de destino, como a versão 1.25.02.
  </Step>
  <Step num="3" title="Execute o Comando de Download">
    No Console Steam, insira o comando de download usando a seguinte sintaxe: `download_depot 454650 454651 [ID_do_Manifesto]`. Pressione Enter para iniciar o download em segundo plano.
  </Step>
  <Step num="4" title="Substitua os Arquivos do Jogo">
    Assim que o console exibir uma mensagem confirmando que o download foi concluído, navegue até o caminho da pasta designada. Copie os arquivos baixados e substitua os arquivos no seu diretório principal.
  </Step>
</Steps>

<Callout type="info" title="Local de Download da Steam">
A Steam baixa os depósitos do console no seu diretório de instalação padrão em `Steam\steamapps\content\app_454650\depot_454651`. Certifique-se de copiar o conteúdo desta pasta, e não a pasta em si, para o seu diretório principal.
</Callout>

</Section>

<Section icon={Swords} title="Instalação de Ferramentas de Modding Essenciais">

Assim que a versão do seu jogo estiver alinhada com o software compatível, você deve configurar a estrutura principal que permite a execução de arquivos personalizados. O ecossistema de modding depende de dois componentes primários: o Eternity Patcher, que ignora as verificações de arquivos do jogo, e o Mod Installer, que lida com a integração de arquivos.

Para garantir que esses utilitários funcionem corretamente, revise as diferenças operacionais específicas e os requisitos das duas principais ferramentas de modding:

| Ferramenta Utilitária | Formato de Arquivo Principal | Dependência Chave | Diretório de Instalação |
|-|-|-|-|
| Eternity Patcher | .ini / .dll | Diretório XV2Patcher | Pasta raiz do jogo |
| XV2INS | .x2m | xv2ins.exe | Qualquer lugar (Link para bin/xinput1_3.dll) |

Para instalar essas ferramentas, extraia o arquivo do Eternity Patcher diretamente no seu diretório raiz. A pasta raiz é onde reside o executável do seu jogo. Em seguida, inicie o aplicativo XV2INS e aponte-o para o executável do seu jogo quando solicitado. Isso estabelece a conexão necessária para gravar novos arquivos no diretório do seu jogo.

<Callout type="tip" title="Faça Backup dos Arquivos de Save Primeiro">
Antes de executar o instalador pela primeira vez, faça uma cópia do seu arquivo de save localizado no diretório Steam Userdata. Alterações na configuração de mods podem ocasionalmente corromper os dados salvos.
</Callout>
