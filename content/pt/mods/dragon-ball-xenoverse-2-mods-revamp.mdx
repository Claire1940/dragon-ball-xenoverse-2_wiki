## Entendendo o Processo de Modding

### Verificação de Compatibilidade
Antes de mergulhar no processo de modding, é crucial garantir que sua versão do jogo seja compatível com o mod que você deseja instalar. Por exemplo, o Revamp pode não ser compatível com todas as versões do Dragon Ball Xenoverse 2. Verifique as notas de compatibilidade ou o site do mod para requisitos específicos.

### Backup do Jogo
Para evitar qualquer perda potencial de dados, é sempre uma boa ideia fazer backup do seu jogo antes de fazer qualquer alteração. Dessa forma, se algo der errado durante a instalação do mod, você pode voltar ao estado original do jogo sem perder nenhum progresso.

## Guia Passo a Passo para Instalar Mods

### Passo 1: Desinstalar Mods Existentes
Se você tiver algum mod instalado, é importante removê-lo para evitar conflitos com o novo mod. Isso inclui excluir os arquivos do mod do diretório do seu jogo e quaisquer pastas associadas.

### Passo 2: Excluir Arquivos do Jogo e Desinstalar
Em seguida, você precisará excluir os arquivos do jogo da biblioteca do Steam e desinstalar o jogo. Isso garante uma instalação limpa sem arquivos leftovers que poderiam causar problemas.

### Passo 3: Reinstalar o Jogo
Depois que o jogo for desinstalado, você pode reinstalá-lo a partir do Steam. Durante o processo de instalação, certifique-se de validar os arquivos para garantir que tudo esteja funcionando corretamente.

### Passo 4: Adicionar o Patcher
Baixe o patcher do mod de uma fonte confiável e extraia os arquivos. Em seguida, navegue até o diretório do seu jogo usando o aplicativo Steam e localize a pasta "Xenoverse 2".

### Passo 5: Copiar Arquivos do Patcher
Copie todos os conteúdos da pasta do patcher para o diretório do seu jogo. Isso incluirá quaisquer arquivos necessários para o mod funcionar corretamente.

### Passo 6: Configurar o Instalador
Abra o instalador e configure-o conforme suas preferências. Você pode instalar mods de personagens ou habilidades aleatórias para testar se o patcher está funcionando corretamente.

### Passo 7: Iniciar o Jogo
Após a configuração do instalador, inicie o jogo para verificar se a versão do patcher é exibida na tela inicial. Se estiver, você pode prosseguir para o próximo passo.

### Passo 8: Extrair o Lazy Bone Save Editor
Baixe o Lazy Bone Save Editor e extraia os arquivos. Este ferramenta é usada para remover IDs não usados e validar flags no seu jogo.

### Passo 9: Remover IDs Não Usados e Validar Flags
Carregue o diretório do seu jogo e o arquivo de save no Lazy Bone Save Editor. Remova quaisquer IDs não usados e valide as flags para garantir que tudo esteja configurado corretamente.

### Passo 10: Baixar e Instalar Revamp
Baixe o Revamp de uma fonte confiável, extraia os arquivos e abra o aplicativo. Navegue até a pasta do diretório do seu jogo e instale o mod.

### Passo 11: Iniciar o Jogo a partir da Pasta Bin
Finalmente, inicie o jogo a partir da pasta bin para garantir que o mod foi instalado corretamente.

Instalar mods pode melhorar significativamente sua experiência em Dragon Ball Xenoverse 2. Ao seguir este guia passo a passo, você pode instalar e usar mods para melhorar seu jogo.
